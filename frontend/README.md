# Yubus Deploy Repo

This is the clean ASCII-only deployment copy of the Yubus project.

## Deploy

- Frontend: GitHub Pages via `.github/workflows/deploy-pages.yml`
- Backend: Render Blueprint via `render.yaml`

## Backend env vars

- `ALLOWED_ORIGIN`
- `DB_URL`
- `DB_USER`
- `DB_PASSWORD`
- `SMTP_USER`
- `SMTP_APP_PASSWORD`
- `SMTP_FROM`
